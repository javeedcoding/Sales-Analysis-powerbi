# Sales Analysis Power BI Dashboard

# Overview

This project analyzes sales data using Power BI to identify sales trends, top-performing countries, product lines, deal sizes, and order statuses.

# Tools Used

- Power BI
- DAX
- SQL
- Python

# Key Insights

- USA generated the highest sales.
- Month 11 recorded the highest sales.
- Classic Cars was the best-performing product line.
- Medium deal sizes generated the most revenue.
- Shipped orders contributed the highest sales.
