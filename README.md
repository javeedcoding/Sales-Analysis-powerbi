# Sales-Analysis-powerbi
Sales Analysis Power BI Dashboard
Project Overview
This project analyzes sales data using Power BI. The dashboard provides insights into sales performance across countries, months, product lines, deal sizes, and order statuses.

# Tools Used
Power BI
DAX
SQL (Data Analysis)
Python (Data Cleaning and Analysis)
# Dashboard Pages
1. Sales Overview
Total Sales
Average Sales
Sales by Country
Sales by Month
2. Sales Analysis
Sales by Product Line
Sales by Deal Size
Sales by Status
# Key Insights
USA generated the highest total sales.
Spain was the second-highest sales contributor.
Month 11 recorded the highest sales.
Classic Cars was the best-performing product line.
Trains generated the lowest sales.
Medium deal sizes generated more revenue than large deal sizes.
Shipped orders contributed the highest sales.
